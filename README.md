## Kurulum

Son versiyonu build etmek için önce nodejs yükleyin, ardından:
```sh
npm install web-ext
web-ext build
```
Ardından Firefox->Eklentiler->Dosyadan eklenti yükle'yi kullanarak yükleyin.
